# dotfiles

``` sh
$ make install # -- Install everything from scratch; link dotfiles
$ make update  # -- Update everything
$ make clean   # -- Uninstall and unlink everything
```

## Environment

* Zsh
* Git
* Homebrew
* Ruby
* Node.js
* Memcached
* MongoDB
* MySQL
* Postgres
* Redis
